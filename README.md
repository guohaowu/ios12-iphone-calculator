# ios12-calculator
<div>使用vue cli3搭建的移动端项目，模仿iOS12系统下的IPhone计算器功能。</div>
<div>本项目适合vue新手入门参考，拷贝项目代码到本地后，直接运行npm run serve就可启动项目。</div>
<div>涉及到的vue知识点有：</div>
1. 列表渲染： v-for
2. Class 与 Style 绑定： v-bind:class; v-bind:style
3. 事件绑定: v-on
4. 计算属性： computed
5. 监听器： watch
